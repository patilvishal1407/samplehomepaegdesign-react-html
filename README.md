## 🚀 Running the Project

This repository contains two versions of the Home Page:
1. React Version
2. HTML Version
---

## 📁 1. React Version

### 📌 Steps to run

```bash
cd ReactApp
npm install
npm run dev
```

### 🌐 Open in browser:

```
http://localhost:5173
```

## 📁 2. HTML Version

### 📌 Steps to run

```bash
cd HtmlApp
```

Then simply open:

```
index.html
```

* Right click on `index.html`
* Click **"Open with Live Server"**

---

## ⚙️ Requirements

* Node.js (v16+ recommended)
* npm
* Browser (Chrome recommended)

---

## 📌 Notes

* React version is dynamic and component-based
* HTML version is static (for quick preview/design reference)
* Both represent the same UI design
